Proyecto 2: Tienda en Línea

Este proyecto es una tienda en línea que permite a los usuarios navegar y comprar productos. La tienda está desarrollada con HTML, CSS y JavaScript, y utiliza una base de datos para gestionar el inventario y los pedidos. El objetivo es proporcionar una experiencia de compra en línea sencilla y eficiente.

  • Catálogo de productos con imágenes y descripciones
  • Carrito de compras y gestión de pedidos
  • Proceso de pago seguro
  • Interfaz de usuario atractiva y responsiva
